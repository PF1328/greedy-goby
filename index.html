Greedy Goby

Jeu rogue-like 2D coopératif

Présentation du projet

Greedy Goby est un jeu de type rogue-like 2D jouable en coopération à deux joueurs en ligne. Le joueur incarne un gobelin cupide explorant un donjon infini afin d’accumuler un maximum d’or. Le jeu repose sur l’exploration de salles générées procéduralement, la coopération entre joueurs et une progression basée sur la prise de risque.

Technologies utilisées

  • Python
  • Pygame
  • Git / GitHub
